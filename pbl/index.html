MintyMind

A web application for Task Planning using DAA

Welcome

What is Our Project About?

  • MintyMind is an aesthetic task planning application that combines beautiful UI with powerful algorithms to optimize your daily productivity
  • It uses advanced sorting, graph traversal, and greedy scheduling algorithms to create intelligent task schedules
  • The platform is designed with a soft, minimal pastel aesthetic and focuses on clear visual hierarchy and ease of use
  • Users can manage tasks, track progress, maintain streaks, and visualize algorithm performance in real-time
Features

Smart Features

  • Merge Sort & Quick Sort: Compare and visualize different sorting approaches for optimal task ordering
  • Graph Analysis: Detect task dependencies, resolve cycles with DFS, and generate valid execution orders with BFS
  • Greedy Scheduling: Automatically optimize your daily schedule by prioritizing high-value tasks within available time
  • Streak Tracking: Maintain motivation with a real-time completion streak counter that rewards consistency
  • Persistent Storage: All data is saved locally using browser storage for privacy and offline access
